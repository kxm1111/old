Maven项目已创建完成！以下是项目结构和功能：

## 项目结构

```
d:\qoder\Project
├── pom.xml                           # Maven配置文件
├── src\main\java\com\example\
│   └── HelloServlet.java            # Servlet处理/hello路径
├── src\main\webapp\
│   ├── WEB-INF\web.xml              # Web配置
│   └── index.jsp                    # 首页JSP
└── target\
    └── hello-webapp.war             # 打包生成的WAR包
```

## 功能说明

1. **index.jsp** - 访问根路径 `/` 时显示：
   - 输出"你好"
   - 显示用户IP地址（使用 `request.getRemoteAddr()`）

2. **HelloServlet** - 访问 `/hello` 路径时显示：
   - 输出"你好世界"

3. **WAR包** - 已成功生成：
   - 文件位置：`target/hello-webapp.war`
   - 可部署到 Tomcat、Jetty 等 Servlet 容器

## 使用方式

将 `hello-webapp.war` 部署到 Tomcat 后：
- 访问 `http://localhost:8080/hello-webapp/` → 显示"你好"和IP地址
- 访问 `http://localhost:8080/hello-webapp/hello` → 显示"你好世界"
